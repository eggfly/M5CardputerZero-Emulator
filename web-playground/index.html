Quick Start

The emulator runs the real APPLaunch firmware in your browser via WebAssembly. No installation required.

Keyboard Controls

Z / Left Arrow  — Switch app left
C / Right Arrow — Switch app right
Enter           — Open selected app
Escape          — Back / Home
Letters/Numbers — Type in apps (CLI, etc.)

Desktop Emulator

For full-featured development, download the native emulator:

# macOS
brew install sdl2 sdl2_image freetype
git clone --recursive https://github.com/eggfly/M5CardputerZero-Emulator.git
cd M5CardputerZero-Emulator && mkdir build && cd build
cmake .. && make -j8
./cardputer-emu

Build Your Own App

Apps are LVGL-based C/C++ projects compiled with the M5Stack Linux Libs SDK:

git clone --recursive https://github.com/eggfly/M5CardputerZero-UserDemo.git
cd projects/UserDemo
scons -j$(nproc)    # SDL2 simulation mode (PC)
./dist/M5CardputerZero-UserDemo

App Architecture

Each app exports void ui_init(void) and is loaded by the emulator as a dynamic library (.dylib / .so / .dll). In the web version, apps are statically linked into the WASM binary.
